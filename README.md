# Program Installer – менеджер установки программ с визуальным интерфейсом

## Описание

**Program Installer** — это десктопное приложение с графическим интерфейсом для удобного отслеживания и управления установкой программ. Приложение загружает список программ из JSON-файла (или с удалённого сервера), отображает их иконки, вес и ссылки для скачивания. Основная цель — помочь пользователю последовательно установить все необходимые программы и отмечать уже установленные.

### Основные возможности

- Загрузка списка программ из локального или удалённого JSON-файла
- Кэширование иконок программ на локальном диске (для быстрого доступа)
- Постраничный просмотр не установленных программ (с кнопками навигации)
- Отображение веса программы в удобном формате (MB / GB)
- Список всех программ с фильтром «Только не установленные»
- Открытие ссылок для скачивания в браузере одним кликом
- Сохранение прогресса – отметка программ как установленных
- Автоматическое обновление прогресса и статистики
- Предзагрузка всех иконок в фоновом режиме
- Современный дизайн с кастомизированными стилями кнопок

## Как это работает

При запуске приложение создаёт необходимые папки (src/icons/, src/json/). Если JSON-файл отсутствует локально, программа скачивает его по указанному URL. JSON парсится, формируется список программ с полями: название, URL иконки, вес установщика, массив ссылок для скачивания, флаг установки. Все иконки скачиваются и кэшируются в src/icons/ в формате PNG.

Интерфейс разделён на два режима:
- Режим установки – постраничный просмотр не установленных программ
- Режим списка – полный список всех программ с фильтрацией

## Требования

- Python 3.7 или новее
- Установленные зависимости:
  - requests – для скачивания JSON и иконок
  - Pillow – для обработки и сохранения изображений
  - tkinter – обычно входит в поставку Python

## Установка и запуск

### Из исходного кода (.py)

1. Клонируйте репозиторий
2. Установите зависимости: pip install -r requirements.txt
3. Запустите .py-скрипт

### Из собранного .exe

1. Запустите ProgramInstaller.exe
2. При первом запуске программа создаст необходимые папки и загрузит JSON

## Инструкция по использованию

### Режим установки (основной экран)

1. Просмотр программы
   - Отображается иконка, название и вес текущей программы
   - Используйте кнопки < и > для переключения между не установленными программами

2. Установка программы
   - Нажмите «Установить (~X MB/GB)»
   - Откроется окно со ссылками для скачивания
   - Кликните по любой ссылке – она откроется в браузере
   - Скачайте и установите программу вручную

3. Отметка об установке
   - После завершения установки нажмите «✔ Установлено»
   - Программа переместится в список установленных
   - Обновится прогресс-бар и статистика

4. Управление прогрессом
   - Кнопка ⟳ – сбросить весь прогресс (отметить все программы как не установленные)
   - Кнопка ☰ – переключиться в режим списка

### Режим списка

1. Просмотр всех программ
   - Отображаются все программы из JSON
   - ✔ – программа установлена
   - ❌ – программа не установлена

2. Фильтрация
   - Включите «Только не установленные» – отобразятся только ❌

3. Установка из списка
   - Дважды кликните / выберите программу в списке
   - Откроется окно со ссылками для скачивания

4. Возврат к режиму установки
   - Нажмите кнопку ⊡ в правом верхнем углу

## Интерфейс и навигация

- ⟳ – Сбросить прогресс (все программы становятся не установленными)
- ☰ / ⊡ – Переключение между режимами (установки / списка)
- < / > – Навигация по не установленным программам
- Установить – Открыть ссылки для скачивания
- ✔ Установлено – Отметить программу как установленную
- Только не установленные – Фильтр в режиме списка

## Важные замечания

- JSON URL: В коде используется заглушка 'ссылка'. Замените её на реальный URL вашего JSON-файла перед использованием.
- Сохранение прогресса: Флаг IsDownloaded не сохраняется между сессиями автоматически. Для сохранения состояния добавьте запись в JSON или используйте отдельный файл конфигурации.
- Кэш иконок: Иконки сохраняются в папку src/icons/. Для очистки кэша просто удалите эту папку.
- User-Agent: Для обхода блокировок некоторыми серверами (например, Wikimedia) используются специальные заголовки.
- Иконки по умолчанию: Если иконка не загрузилась, отображается текст «🖼️ Нет иконки».

## Возможные проблемы и решения

- JSON не загружается – Проверьте URL в коде и интернет-соединение
- Иконки не отображаются – Некоторые серверы блокируют автоматические запросы. Иконка сохранится при следующем запуске (кэш)
- Ошибка PIL.UnidentifiedImageError – Убедитесь, что URL иконки ведёт на изображение, а не на HTML-страницу
- Кнопки не реагируют – Проверьте, что все зависимости установлены (pip install requests Pillow)
- Программа зависает при загрузке – Иконки загружаются в фоновом режиме – подождите несколько секунд

## Пример работы

1. Запуск → загрузка JSON → отображение первой не установленной программы
2. Нажатие «Установить» → открывается окно со ссылками:
   "Чтобы установить программу, перейдите по ссылкам ниже:" и список ссылок
3. Пользователь скачивает и устанавливает программу вручную
4. Нажатие «✔ Установлено» → программа уходит в список установленных
5. Прогресс обновляется → «Готово: 1/10 (10.0%)»
6. Автоматический переход к следующей программе

## Настройка под себя

Изменение URL JSON

В файле main.py найдите строку:
self.json_name = "src/json/programs.json"
и замените на ваш URL (например, https://example.com/programs.json).

Изменение стилей кнопок

Стили настраиваются в методе setup_styles():
- Green.TButton – зелёная кнопка (✔ Установлено)
- Blue.TButton – синяя кнопка (Установить)
- WhiteBlue.TButton – белая кнопка с синим текстом (⟳, ☰, ⊡)
- TransparentBlue.TButton – прозрачная кнопка с синим текстом (<, >)

Изменение размера иконок

В методе _load_icon_from_file() измените размер:
img = img.resize((256, 256), Image.LANCZOS) – вместо 256 укажите нужное значение
